>>コレ is 何
デスクトップマスコットです。

>>使い方
Desktop_Mascot.exeが入っているディレクトリにマスコットにしたいpng画像を「mascot.png」を置いてください。
透過情報があれば非矩形ウィンドウになります。
初期マスコットはプロ生ちゃんです。

起動後、左下にマスコットが出るはずです。
・左ドラッグで移動
・マウスホイールクリックで位置保存
・右クリックで保存位置に移動
・Escで終了
